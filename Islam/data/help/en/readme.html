Islam Program - User Guide

Introduction:

The Islam Program is a multi-functional application designed to facilitate the practice of Islamic worship and enhance spirituality for users. The program offers a wide range of useful features including electronic Tasbeeh, Azkar reminders, prayer times, Quran reading, and much more.

Main Features:

  1. Electronic Tasbeeh:

    The program provides an electronic Tasbeeh that enables the user to record and monitor the number of Tasbeehs. - Access the Tasbeeh by clicking on the "Tasbeeh" tab.

  2. Azkar Reminders:

    The program offers periodic reminders of Azkar to help remind the user of important Azkar throughout the day.

  3. Prayer Times:

    Users can know the prayer times in any country.

  4. Quran Reading:

    The program provides Quran reading and listening to it by verses, pages, parts, quarters, chapters, and parties. - Users can customize the display of the Quran and use shortcuts to navigate quickly through the text.

  5. Adhan:

    The program alerts the user to the prayer time and also before the prayer time, and it also allows the user to customize the muezzin's voice by changing the muezzin's voice and placing it in the path.

    Path: data\sounds\adaan

  6. Quran Interpretation:

    The program provides several interpretations of the Holy Quran including:

    1. Al-Baghawi
    2. Al-Qurtubi
    3. Al-Saadi
    4. Al-Tabari
    5. Al-Tafsir Al-Muyassar

Shortcuts:

In the "Tasbeeh" tab:
- c: to know the number of Tasbeehs.

In the Quran open window:
- down: next verse.
- up: previous verse.
- left: read the current verse.
- space: listen to the verse.
- alt+write: next.
- alt+left: previous.
- ctrl+space: play to the end.
- ctrl+shift+g: go to verse.
- Ctrl+g: go to.
- Ctrl+windows+shift+h: to show or hide the window.

Important Note:

When placing the muezzin's sound file in the above-mentioned path, please write the file name in English and the file extension should be .wav

We hope you have an enjoyable and beneficial experience with the Islam Program.
